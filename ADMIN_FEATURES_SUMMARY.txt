     ├─ Show User Details: Name, Email, Phone, Address

     ├─ Show Package Details:
     │  ├─ Custom places

  │  └─ Creating custom package requests
  │

  │  ├─ Click to browse files

  │
  ├─ Supported Formats:
  │  ├─ PNG
  │  ├─ JPG

  │
  ├─ Features:

  │
  └─ Admin View:
     ├─ Image preview on package cards
     ├─ Full-size image for inspection
     └─ Helps with approval decisions
